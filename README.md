# 🎧 Spotify UI Clone

A responsive **Spotify Web Player UI Clone** built using **HTML5 and CSS3**.  
This project recreates the visual layout of Spotify’s web interface for learning and front-end practice purposes.

🔗 **Live Demo:**  
https://tarunish.github.io/Spotify_clone/

---

## 📌 About The Project

This project focuses on replicating the user interface of Spotify using only HTML and CSS.  
It is a static front-end project designed to improve layout design skills and understand real-world UI structuring.

The design includes:

- Sidebar navigation  
- Playlist sections  
- Music cards layout  
- Footer music player UI  

> ⚠️ This is a static UI clone created for educational purposes only. It does not include real music playback or backend functionality.

---

## 🚀 Features

- Spotify-inspired modern dark theme  
- Sidebar with navigation options  
- Playlist and trending music sections  
- Styled footer music player layout  
- Responsive design structure  
- Clean and organized code  

---

## 🛠️ Built With

- **HTML5** – Structure  
- **CSS3** – Styling & Layout  

No JavaScript used.
